About the Game

This is a modern take on the classic snake game, designed to be simple, intuitive, and engaging. Players control a snake that grows longer each time it eats food, while avoiding collisions with walls or its own body.

The game runs inside a fixed grid and supports real-time movement using arrow keys or WASD. It includes a dynamic score tracker that updates during gameplay and records each player's score at the end of every round.

Players can pause and resume the game at any time, and restart after losing. Additional features include customizable game speed, grid size, snake and food appearance, and optional grid visibility.
